## `useBottomSheet`

`BottomSheet` component is controlled by `UseBottomSheetReturn`(the return type of `useBottomSheet`), so they go hand-in-hand.
Check out the [UseBottomSheetReturn](/apis/React/UseBottomSheetReturn/Methods) for use cases and examples.

### Interface

```ts
useBottomSheet: (props: UseBottomSheetProps) => UseBottomSheetReturn;
```

### Example
