## Submit Your Project

If you've built something awesome with PlainSheet, we'd love to showcase it! Here's how you can get featured:

1. **Open an issue** on our [GitHub repository](https://github.com/plainsheet/plainsheet)
2. **Include details** about your project:
   - Project name and description
   - Live URL (if available)
   - Brief explanation of how you're using PlainSheet
3. **Tag it** with the `showcase` label

We review submissions regularly and will add projects to this page.

## What We're Looking For

- **Production applications** that use PlainSheet
- **Open source projects** that others can learn from
- **Commercial applications** that demonstrate real-world usage
- **Or any other interesting usage** of PlainSheet
