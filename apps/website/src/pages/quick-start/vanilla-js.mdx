# Quick Start Guide

## Installation

### Using NPM

```sh npm2yarn
npm install @plainsheet/core
```

### Using CDN

```html copy
<script src="https://cdn.jsdelivr.net/npm/@plainsheet/core@latest/dist/plainsheet-core.umd.js"></script>

// Or, via unpkg
<script src="https://unpkg.com/@plainsheet/core@latest/dist/plainsheet-core.umd.js"></script>
```

## Getting Started

For all the customizable options, check out [the API document](/apis/vanilla-js)!

### Using NPM

<VanillaCodePlayground
  code={`
import { createBottomSheet } from "@plainsheet/core";

const bottomSheet = createBottomSheet({ content: 'text' });

bottomSheet.mount();
bottomSheet.open();

// Change the content, on the fly.
bottomSheet.props.content = '<div>New Content <br/><br/><br/> </div>';

`}
/>

### Using CDN (UMD Module)

```ts copy
const bottomSheet = window.BottomSheet.createBottomSheet({
  content: ``,
});
// Rest is the same as the NPM version
```
