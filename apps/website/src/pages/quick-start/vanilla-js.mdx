# Quick Start Guide

## Installation

### Using NPM

```sh npm2yarn
npm install @plainsheet/core
```

### Using CDN

```html copy
<script src="https://cdn.jsdelivr.net/npm/@plainsheet/core@latest/dist/plainsheet-core.umd.js"></script>

// Or, via unpkg
<script src="https://unpkg.com/@plainsheet/core@latest/dist/plainsheet-core.umd.js"></script>
```

## Getting Started

For all the customizable options, check out [the API document](/apis/vanilla-js)!

### Using NPM

<VanillaCodePlayground
  code={`import { createBottomSheet } from "@plainsheet/core";

const bottomSheet = createBottomSheet({ content: "text" });
bottomSheet.mount();

addEventListener("DOMContentLoaded", (event) => {
  createButton("open-button", "Open it!", bottomSheet.open)

  createButton("content-update-button", "Change the content!", () => {
  bottomSheet.props.content = "<div>New Content <br/><br/><br/> </div>";
  })
});

function createButton (id, text, onClick) {
  const btn = document.createElement("button");
  btn.setAttribute("id", id);
  btn.textContent = text;
  window.document.body.appendChild(btn);
  document.querySelector("#" + id).addEventListener("click", onClick);
}
`}
/>

### Using CDN (UMD Module)

```ts copy
const bottomSheet = window.BottomSheet.createBottomSheet({
  content: ``,
});
// Rest is the same as the NPM version
```
