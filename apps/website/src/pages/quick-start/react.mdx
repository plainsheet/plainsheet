# Quick Start Guide

## Installation

```sh npm2yarn
npm install @plainsheet/react
```

## Getting Started

Call the `useBottomSheet` and pass the props to the `BottomSheet`.  
It's that simple, and you have a full control over the bottom sheet via the `instance` and `props`.  
For all the customizable options, check out [the API document](/apis/react)!

```ts copy
import { useState } from "react";
import { BottomSheet } from "@plainsheet/react";

export default function Home() {
  const bottomSheet = useBottomSheet({
    props: {
      // `BottomSheetCore` props
    }
  });

  return (
    <section>
      <BottomSheet
        {...bottomSheet.props}
      >
        Your content goes here
      </BottomSheet>

      <button onClick={bottomSheet.open}>
        Open
      </button>
    </section>
  );
}
```
